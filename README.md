## 🚀 快速启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 启动生产服务器
npm run dev:prod
```

服务将在 `http://localhost:9000` 启动，Swagger 文档访问地址：`http://localhost:9000/api-doc/`
